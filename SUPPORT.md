# Support

This project is maintained as open-source infrastructure for verifiable agent execution evidence.

## Community support

Use GitHub Issues for:

- reproducible bugs
- installation problems
- validator CLI issues
- documentation problems
- minimal examples that do not work as documented

For design discussions, use GitHub Issues unless GitHub Discussions are enabled later.

## Before opening an issue

Please include:

- operating system
- Python version
- package version
- exact command used
- minimal receipt/evidence example if safe to share
- full error message

Do not post secrets, private keys, API tokens, confidential evidence bundles, or production receipts in public issues.

## Scope of unpaid support

There is no guaranteed response time for unpaid support.

This project validates execution evidence for tool invocations. It does not certify legal compliance, prove semantic correctness of tool outputs, or replace sandboxing, IAM, access control, monitoring, or human approval.
